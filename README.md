## 📖 Overview

AndroidOps is an Android Studio project demonstrating background services,
Telegram-based command communication, and Android permission handling for
authorized environments and educational purposes.

---

## ✨ Features

- 📱 Device Status
- 📍 Location
- 🎙️ Microphone
- 📷 Front / Back Camera
- 👥 Contacts
- 💬 SMS
- 📂 File Manager
- 🔄 Foreground Service
- 🚀 Auto Start
- ⚡ Background Service

---

## 📋 Requirements

- Android Studio
- Android SDK
- Java 8+
- Gradle

---

## 🚀 Installation

```bash
git clone https://github.com/USERNAME/AndroidOps.git
```

Open in Android Studio.

Sync Gradle.

Build the project.
